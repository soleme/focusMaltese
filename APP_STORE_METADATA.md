## Description
집중해 말티즈는 말티즈 요정과 함께 집중 시간을 관리하는 감성형 포커스 앱입니다.

현재 버전에서는 아래 기능을 사용할 수 있습니다.
- 25분 집중 / 50분 몰입 프리셋
- 집중 시작, 일시정지, 재개, 성공, 실패 흐름
- 세션 완료 시 XP와 개껌 보상
- 레벨업과 누적 성장 확인
- 최근 세션 기록과 주간 집중 흐름 확인
- 알림, 사운드, 진동 설정

집중해 말티즈는 복잡한 업무 도구보다, 부담 없이 집중 리듬을 유지하도록 돕는 데 초점을 맞춥니다.
하루의 작은 집중을 쌓아 말티즈와 함께 성장해보세요.
