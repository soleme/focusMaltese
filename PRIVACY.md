# Focus Maltese Privacy Notes

## Summary
Focus Maltese는 현재 버전에서 사용자 데이터를 기기 로컬에만 저장합니다.
서버 계정, 클라우드 동기화, 광고 추적, 외부 분석 SDK는 포함하지 않습니다.

## Data Stored On Device
- 레벨
- 경험치(XP)
- 개껌 수량
- 최근 집중 기록
- 마지막 세션 상태
- 앱 설정 값
  - 사운드
  - 진동
  - 알림 사용 여부

## Data Not Collected
- 이메일
- 전화번호
- 위치
- 연락처
- 사진 라이브러리
- 광고 식별자
- 사용자 행동 분석 이벤트 전송

## Notifications
- 로컬 알림만 사용합니다.
- 사용자가 iOS 권한을 허용한 경우에만 집중 종료 알림을 표시합니다.

## Data Deletion
- 앱 내 `진행 데이터 초기화` 기능으로 로컬 데이터를 삭제할 수 있습니다.
- 앱 삭제 시 기기 내 로컬 데이터도 함께 제거됩니다.

## Future Changes
- 계정 연동, 클라우드 저장, 외부 분석 도구가 추가되면 이 문서를 반드시 갱신해야 합니다.
